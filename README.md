# Five Feautures of this website

1. Login & Registration process is here. User can register with his name, photo url, email and password.
2. Registered user can only log in with their emails & password.
3. Login with google is also there.
4. In user registration, password is validated by these conditions:<br/>
    i. Password must be at least 6 characters long. <br/>
    ii. Password must contain at least one capital letter & one special characters.
5. Only logged in users can see event service details, book an event and can browse his booked events.
